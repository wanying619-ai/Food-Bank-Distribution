# Food Bank Distribution App

## Description
This project is created to practice basic web development concepts using PHP, HTML, CSS and JavaScript.

The application supports 3 types of users:

1. **Users** can register and log in to access food bank services, manage their profiles, view nearby food bank locations and check available food categories.
2. **Shops** can register and log in to manage their shop profiles and the food categories they provide.
3. **Administrators** can log in to manage the system, including maintaining food bank contents, managing users and shops and viewing overall reports.

## Technologies Used
- PHP  
- HTML  
- CSS  
- JavaScript  
- MySQL  
- XAMPP  

## How to Run
1. Download and install **XAMPP**.
2. Start the **Apache** and **MySQL** modules.
3. Click **Admin** on the MySQL module to open **phpMyAdmin**.
4. Click **New** and create a database named `database`.
5. Import all 8 SQL files into the database.
6. Place the `asg` folder inside the `xampp/htdocs` directory.

### Access the Application
**User / Shop:**
Type “http://localhost/asg/MainPage.html” in Chrome.

**Admin:**
Type “http://localhost/asg/AdminMainPage.html” in Chrome.
